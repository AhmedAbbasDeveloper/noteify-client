# Noteify Client

This repo contains the client for [Noteify](https://noteify-app.netlify.app), a distributed note taking app. This client makes requests to the [API](https://github.com/AhmedAbbasDeveloper/noteify-server) so users can add and remove notes.

The project uses the following tech stack:
- TypeScript
- React
- GraphQL/Apollo
- Material UI
- Netlify
